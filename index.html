Interactive Cubic B-Spline Curve Fitting

Fit a smooth cubic B-spline through clicked or file-loaded data points.

C program that allows the user to click points or load them from a file and then fits a smooth cubic B-spline curve through the data. I implemented Gaussian elimination with partial pivoting to solve the linear system for the spline coefficients and used FPToolkit to render the curve and control points.

Tools: C, FPToolkit, Gaussian elimination, cubic B-splines

Neural network from scratch to classify 28x28 pixel digits.

Built a two-layer multilayer perceptron from scratch in Python to classify MNIST digits. I implemented forward propagation, backpropagation with a sigmoid activation, and momentum-based gradient updates, then plotted train/test accuracy and generated a confusion matrix to analyze performance.
