# Angel Lores - Portfolio

CS 563 Intro to Web Development - Final Project

- Live site: https://www.angel-lores.com
- GitHub repo: https://github.com/angel-lores/portfolio

## Overview

Single page portfolio site with a fixed navbar and four sections (About, Experience, Projects, Contact)

## Technologies

- HTML5
- CSS3 + media queries
- JavaScript (ES6)
- Bootstrap 5 (carousel)
- Formspree for handling contact form submissions
- Deployed with GitHub Pages and a custom domain

## Features

- **Navbar** - Fixed at the top, uses smooth scrolling to jump between sections.
- **About** - Intro text, photo, and brief professional background.
- **Experience** - Three internship roles displayed as cards with logos and descriptions.
- **Projects** - Bootstrap carouself with three projects, each with an image, description, and GitHub link.
- **Contact** - Form wired to Formspree with required fields and a JS character counter + minimum length check.

## How to Run

1. Clone the repo

```bash
git clone https://github.com/angel-lores/portfolio.git
cd portfolio
```

2. Open `index.html` in a browser, or serve locally with a simple HTTP server
